# 界面线框图

> 以下线框图定义了主要布局和功能区域。视觉设计不做限定——你可以按自己的审美和选用的 UI 库来实现，只需包含线框图中的功能区域。

---

## 1. 整体布局

```
┌─────────────────────────────────────────────────────────────────┐
│  🏢 星汇智慧空间          设备看板 │ 工单管理          [用户头像] │
├────────────┬────────────────────────────────┬───────────────────┤
│            │                                │                   │
│  左侧边栏   │         主内容区                │    AI 助手        │
│  (楼栋选择)  │                                │   (聊天侧栏)      │
│            │                                │                   │
│  ┌────────┐│                                │                   │
│  │ B1 栋 ◄││                                │                   │
│  │ B2 栋  ││                                │                   │
│  │ B3 栋  ││                                │                   │
│  │ B4 栋  ││                                │                   │
│  │ B5 栋  ││                                │                   │
│  └────────┘│                                │                   │
│            │                                │                   │
│  状态筛选   │                                │                   │
│  ● 全部    │                                │                   │
│  ○ 正常    │                                │                   │
│  ○ 告警    │                                │                   │
│  ○ 故障    │                                │                   │
│  ○ 离线    │                                │                   │
│            │                                │                   │
├────────────┴────────────────────────────────┴───────────────────┤
│  AI 助手聊天按钮（点击展开/收起右侧栏）                            │
└─────────────────────────────────────────────────────────────────┘
```

- 左侧边栏固定宽度（约 200px），楼栋选择 + 状态筛选
- 主内容区自适应宽度
- AI 助手默认收起，点击按钮展开（约 360px 宽）
